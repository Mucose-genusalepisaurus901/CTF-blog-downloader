## 🧰 What this app does

CTF-blog-downloader helps you save useful CTF blog posts for offline use. It can turn writeups, bug analysis posts, and tool notes from sites like 先知社区, 博客园, and CSDN into Markdown files.

You can then open those files in Obsidian, Typora, or any other note app. This makes it easier to search, sort, and read your notes when you do not have internet access.

## ✨ What you can do

- Save online blog posts as Markdown files
- Keep writeups in one local folder
- Read posts offline in note apps
- Build a personal knowledge base for CTF practice
- Collect notes from several common blog sites
- Store content in a format that is easy to search

## 🚀 How to use it

1. Start the app.
2. Paste the blog post link into the input box.
3. Choose the save folder on your computer.
4. Click the button to start the download.
5. Wait while the app gets the page and converts it into Markdown.
6. Open the saved file in Obsidian, Typora, or Notepad.

If you want to collect many posts, repeat the same steps for each link.

## 🔧 Common problems

### The app does not open

- Make sure you downloaded the Windows file
- Check that the download finished
- Try opening the file again from the folder where you saved it

### The page does not convert well

- Use the full article link, not a short link
- Try another post from the same site
- Some pages use extra layout blocks that may affect the output

### Images do not show

- Check that the image files were saved in the same folder
- Move the Markdown file and image folder together
- Open the file in a note app that supports local images

### The file cannot be found

- Search your Downloads folder
- Check the save path in the app
- Make sure the output process finished before closing the app
